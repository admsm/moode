moOde - camilladsp default supplied configurations

ash-ir R02-control room.yml - Demonstration of  Binaural Room Impulse Responses R02 Control Room of the ASH-IR dataset. Config is without HpCF. (https://github.com/ShanonPearce/ASH-IR-Dataset)
crossfeed bs2b.yml          - Bs2b implementation for CamillaDSP from Wang-Yue. Config contains 5 setting sets wang-yue for more info. (https://github.com/Wang-Yue/camilladsp-crossfeed)
crossfeed mpm.yml           - Implementation of Mikhail Naganov 'reverse engineered' SPL Phonitor Mini Crossfeed with DSP by Ebert-Hanke (https://github.com/Ebert-Hanke/autoeq2camilladsp)
crossfeed natural.yml       - Implementation by Ebert-Hanke (https://github.com/Ebert-Hanke/autoeq2camilladsp)
crossfeed pow_chu_moy.yml   - Implementation of Pow Chu Moy Crossfeed based on Linkwitz by Ebert-Hanke (https://github.com/Ebert-Hanke/autoeq2camilladsp)
flat.yml                    - Empty config
ms-matrix narrow.yml        - m-s matrix with side signal reduce
ms-matrix wide.yml          - m-s matrix with side signal increased
peq 10 band.yml             - generic example of a 10 band parametrice equalizer
trifield 3ch.yml            - Implementation of Michael Gerzon's Trifield decoder. Requires at least 3 channels available for output. (https://www.meridian-audio.info/public/trifield%5B2563%5D.pdf)
